# Riddles SPA

Simple single-page app with kids-friendly riddles in 3 languages:

- English
- French
- Italian

The app is fully static and lives in `index.html`.
I hope you can enjoy it with your kids.

## Run locally

Open `index.html` in your browser.

`https://rcocetta.github.io/riddles/`
